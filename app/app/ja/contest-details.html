    作品募集期間
    2020年2月1日-2020年4月30日

    ネイチャーズベストフォトグラフィーアジア 2020 ガイドライン

    Nature’s Best Photography ASIAはプロ、アマを問わず、世界中のカメラマンが参加出来る、アジア50カ国の自然にフォーカスをあてた写真コンテストです。過去に撮影した写真、発表済みの作品も対象。撮影地はアジア圏内に限定されます。優秀作品および選択された作品約20点が大判印刷にてアメリカの“Wonders of Wildlife” ワンダーズ オブ ワイルドライフ博物館に年間展示され、アジアの自然を世界に発信することができます。またアメリカ国内、アジア、日本にて巡回展を開催いたします。

    応募期間 2020年2月1日〜2020年4月30日
    撮影地

    アジア50カ国

    Afghanistan / Armenia / Azerbaijan / Bahrain / Bangladesh / Bhutan / Brunei / Burma / Cambodia / China / Georgia / Hong Kong / India / Indonesia / Iran / Iraq / Israel / Japan / Jordan / Kazakhstan / Korea, South / Kuwait / Kyrgyzstan / Laos / Lebanon / Macau / Malaysia / Maldives / Mongolia / Nepal / Oman / Pakistan / Palestine / Philippines / Qatar / Russia / Saudi Arabia / Singapore / Sri Lanka / Syria / Taiwan / Tajikistan / Thailand / East Timor / Turkey / Turkmenistan / United Arab Emirates / Uzbekistan / Vietnam / Yemen
    撮影対象 アジア圏内で撮影された自然写真アジア地域で撮影された自然を被写体とします。過去に撮られた写真も対象となります。

    ワイルドライフ

    野生動物（哺乳類、爬虫類、両生類など−鳥は鳥部門へ）アジアで生息する野生動物に限る。

    風景

    アジアの壮大な景色や展望、広大な風景を捉えたもの。自然生息地での花、木などの植物も含む。

    海

    アジアの海で撮影された海上、海中、海の周りの景色、サンゴ、海中生物、波、海氷、海に関連する生物など。

    鳥

    領土戦い、ネスティング、給餌、交尾の儀式、子育て、またはその他の行動などの野鳥の姿を捉えたもの。

    スモールワールド

    クロースアップで撮影された植物、動物、昆虫、マクロレンズで撮影されたものなど。

    動画

    自然全般を捉えた映像を3分以内に収め、YoutubeやVimeo、またはその他の動画共有サイトにアップし、リンクを応募フォームよりお送り下さい。字幕や音声を編集する場合には英語のみ使用可能。

    ジュニア

    18歳以下のジュニア（小、中、高校生）が対象。対象部門は一般と同じく、風景、ワイルドライフ、海、鳥、スモールワールド、動画。

    賞 グランプリ：1名　賞金$1,000USD 、授賞楯、米国ワンダー オブ ワイルドライフ博物館に年間展示およびネイチャーズベストマガジン掲載。
    部門最優秀賞：7名（各部門1名）授賞楯、米国ワンダー オブ ワイルドライフ博物館に年間展示およびネイチャーズベストマガジン掲載。
    優秀賞：各部門5名　優秀賞から選ばれた数点は、米国ワンダー オブ ワイルドライフ博物館に年間展示。すべての優秀賞作品はネイチャーズベスト フォトグラフィー アジア ホームページギャラリー掲載。
    授賞式：受賞者はアメリカで行われるアワードセレモニー（授賞式）へご招待。（旅費、宿泊費は各自ご負担頂きます）。
    出版：グランプリ及び部門最優秀賞受賞作品は世界中にて購読部数25,000部のNature’s Best Photography誌に掲載されます。
    エントリー費 部門関係なく1口20点までUSD＄25 (日本円約2800円)。応募口数に制限はありません。
    展示 優秀作品約20点は米国ミズリー州にある（位置する）ワンダーズ オブ ワイルドライフ博物館 “Wonders of Wildlife”のネイチャーズベストフォトグラフィー展示場に大判印刷にて年間展示されるので、アジアの自然を世界に発信する機会となります。アメリカ国内、アジア、日本にて巡回展を開催いたします。
    応募 オンライン入稿のみ。WEBサイトからエントリー。
    著作権 写真家は作品の著作権を所有し、自身の自由意志のもと参加するコンテストです。当コンテストの受賞条件として、写真家は「ネイチャーズベストフォトグラフィー出版」(NBP)およびNature’s Best Photography Asia (NBPA)がマガジンに掲載すること、複数展覧会での展示、NBPおよびNBPAのホームページ掲載、カレンダー、ポストカード、その他NBPおよびNBPAコンテストプロモーションの目的で世界中での使用を許可し同意するものとします。NBPおよびNBPAは受賞作品の販売制限をしません。写真家は自身の受賞作品の販売を自由にすることが出来ます。NBPのコンテスト受賞作品は高い評価を得るため、受賞後は売上向上につながる写真家も多くいます。

    オンラインにて作品のアップロード

    オンラインアップロードについて

    作品をオンラインにてアップロードする際、低解像度JPEG(72ppi) 500K以下のコピーを作成して下さい。それぞれのファイルにはあなたの氏名、作品タイトル、部門名をファイル名として保存して下さい。（例：TaroYamada_monkey_wildlife.jpg）
    以下の案内にそってご応募下さい。
    オンライン入稿:
    最終審査では高解像度のファイルをご提出頂きます。（可能であればRAWファイルも）撮影の際、カメラ設定をRAW/Jpegのフォーマットにて撮影されると良いでしょう。また同時にキャプション、撮影情報も最終審査前にご提出頂きます。これらがご提出出来ない場合は最終審査が受けられない可能性がございます。高解像度の提出方法については最終審査に残った作品の応募者にご連絡いたします。

    ルール

    写真は全てファインダーで実際に捉えられたイメージでなければなりません。デジタルスポットやダストスポットを取り除く以外の削除やものを加えることは出来ません。通常範囲でのRAWファイルのプロセスや色やコントラストの調整、および必要最低限の範囲でのクロップピングは可能です。HDRフォトスタッキングおよびパノラマスティッチングに関しては、どのようなテクニックを使用したかの説明を応募時に加えて下さい。
