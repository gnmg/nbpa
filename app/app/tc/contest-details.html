    2020 年亞洲最佳自然攝影大獎指南

    亞洲特優自然攝影大獎誠摯邀請所有的攝影師參加我們第三屆的年度大賽。不論您是住在哪裡，只要是在亞洲拍攝的影像，即可參加。請參閱下列地圖及國家說明以確保您的攝影作品符合資格。由「亞洲最佳自然攝影大賽」所精選的一系列優勝作品將在十月份於華盛頓特區「美國國立自然史博物館」（世界上最廣受尊崇且擁有高訪客數的博物館之一）的「年度最佳自然攝影展」中展出。

    募集期間 201年1月15日 − 2020年4月10日
    攝影地點

    Asian Countries (50 countries)

    Afghanistan / Armenia / Azerbaijan / Bahrain / Bangladesh / Bhutan / Brunei / Burma / Cambodia / China / Georgia / Hong Kong / India / Indonesia / Iran / Iraq / Israel / Japan / Jordan / Kazakhstan / Korea, South / Kuwait / Kyrgyzstan / Laos / Lebanon / Macau / Malaysia / Maldives / Mongolia / Nepal / Oman / Pakistan / Palestine / Philippines / Qatar / Russia / Saudi Arabia / Singapore / Sri Lanka / Syria / Taiwan / Tajikistan / Thailand / East Timor / Turkey / Turkmenistan / United Arab Emirates / Uzbekistan / Vietnam / Yemen
    對象 亞洲地區拍摄的大自然攝影作品。包括過去拍摄的攝影作品。

    野生動物

    野生動物（哺乳類、爬蟲類、兩棲類等−鳥類前往鳥類部門）僅限於棲息在亞洲的野生動物。p>

    野生動物

    捕捉亞洲壮美的景色以及遠眺、遼闊風景的攝影作品。包括大自然生息的花草、樹木等植物。

    海

    亞洲海域為主題拍摄的海上、海中、大海周圍的景色、珊瑚礁、海洋生物、波浪、海洋冰山、與大海有關的生物等攝影作品。

    鳥

    領地争奪、築巢、餵食、交尾、育雛以及其他行為等，捕捉到的野鳥生活的攝影作品。

    微觀世界

    特寫技法拍摄的植物、動物、昆虫、微距拍摄的攝影作品等。

    视频

    高分辨率视频已经成为记录和分享荒野景象的一种重要工具。除了静止的摄影作品以外，“亚洲最佳自然摄影”大奖赛的编辑也希望通过录像制作人和电影制作人的眼睛一览自然风光以及户外体验。告诉我们您的故事，分享您最喜爱的体验，包括您或其他人身处野外时的片段或者您所观察到的野生生物的行为。通过 YouTube、Vimeo 或者其他视频观看平台上传您的作品并将链接（如果需要的话，包括密码）提供给我们。

    獎項 大獎：US $1,000，獎牌，且照片將在史密森尼自然歷史博物館展出。
    二獎 x 5： US $ 500，獎牌，且照片將在史密森尼自然歷史博物館展出。

    邀請 獲獎者將被邀請出席在「史密森尼國立自然史博物舘」舉辦的颁獎典禮。旅費及住宿費獲獎者本人承担

    在線上傳作品

    關於在線上傳

    在線上傳作品時，請製作為低清晰度JPEG(72ppi) 500K以下的復印件。每份文件以您的姓名、作品題目、部門名称作為文件名進行保存。（例如：TaroYamada_monkey_wildlife.jpg）請按照以下說明參加。

    在線提交作品:
    在最終評審階段需要您提供高清晰度文件（可能包括RAW文件）。拍摄時，相機設定在RAW/Jpeg格式化狀態下拍摄較好。并且，在最終評審前，還需要您提供字幕、拍摄參數。不能提供以上資訊時，將可能不能進行最終的評審。關於高清晰度文件的提交方法，請與作品進入最終評審階段的參賽者聯係。

    規定:

    照片必須完全是通過取景镜頭拍摄的影像。除消除數碼點或聚塵點外，不能進行任何刪除以及添加。在通常範圍内，可以對RAW文件進行加工、色彩對比度的調整、以及最小必要限度範圍内的切頭(遮幅效果處理)。對於HDR照片折曡和全景照片縫和，請在提交作品時說明應用了哪種技術手段。不按照本次大賽的規定的攝影作品將從參賽作品中刪除。請選擇合適的部門。審查過程中，可通過審查員的判断變更部門。

    IMPORTANT NOTES

    動物園和水族舘拍摄的作品不属於參賽作品範圍。照片必須完全是通過取景镜頭拍摄的影像。除消除數碼點或聚塵點外，不能進行其他刪除以及加入本来没有拍摄到的物體的行為。在通常範圍内，可以進行RAW文件的加工、色彩對比度的調整、以及最小必要限度範圍内的切頭(遮幅效果處理)。對於HDR照片折曡和全景照片縫和，請在提交作品時說明應用了哪種技術手段。參賽者不能有侵犯其他攝影家著作權、違反規定、或者侵犯私有地、傷害野生生物、破壞環境等行為。不遵守大賽規定的作品將取消參賽資格。
